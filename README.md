# Esep Landing Page

Лендинг для **Esep** — автопилота налоговой отчётности для бизнеса Казахстана.

## Структура

```
landing.html    — основная страница (single-file, всё встроено)
assets/         — изображения и фоны
favicon.png     — иконка
```

## Запуск

Просто откройте `landing.html` в браузере или разверните на любом статическом хостинге.
